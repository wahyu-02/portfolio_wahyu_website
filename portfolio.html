Evaluating CLIP Model for Image & Text Classification

Exploring CLIP's performance in zero-shot image-text classification for livestock species, behavior, location, and time.

Project Overview

This project evaluates CLIP's multimodal model performance in classifying livestock species, behaviors, locations, and time using four variants: RN50, ViT-B/16, ViT-B/32, and ViT-L/14.

Using a zero-shot learning approach, predictions are based on cosine similarity between image features and descriptive text embeddings. The dataset includes 279 livestock images across cows, goats, and sheep.

Results show ViT-L/14 achieves the highest accuracy (84%) for species classification, while ViT-B/16 maintains balanced accuracy across all categories.

Recommendations include fine-tuning CLIP on livestock-specific data and exploring few-shot learning for improved accuracy.

NLP Text Classification - AGNews

Comparing FastText, Transformer, and LSTM for text classification using GloVe embeddings.

Project Overview

This project evaluates the performance of three NLP models—FastText, Transformer, and LSTM—for text classification on the AGNews dataset, using pre-trained GloVe embeddings.

- FastText: Efficient and computationally light, achieving 83% accuracy.

- Transformer: Captures word relationships with self-attention, achieving 90% accuracy.

- LSTM: Handles long-term dependencies but requires more computational resources, achieving 84% accuracy.

ARCH & GARCH Model for Climate Data

Time-series volatility modeling using ARCH & GARCH for climate data analysis.

Project Overview

This project analyzes the volatility of global temperature and CO2 emissions using ARCH & GARCH models. Data sources include Global Carbon Atlas (GCAG) and NASA’s GISTEMP dataset.

ARCH tests reveal significant conditional heteroskedasticity in residuals, indicating the need for more sophisticated volatility models.

GARCH(1,1) was implemented for both GCAG and GISTEMP data with an ARIMA(2,0,1) mean model. Model evaluation shows well-fitted residuals with no significant autocorrelation or heteroskedasticity.
